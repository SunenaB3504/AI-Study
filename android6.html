Chapter 6: Data Storage

Overview of Data Storage

Android provides several options for storing data: SharedPreferences, Internal/External Storage, and databases like SQLite and Room.

SharedPreferences

SharedPreferences prefs = getSharedPreferences("MyPrefs", MODE_PRIVATE);
SharedPreferences.Editor editor = prefs.edit();
editor.putString("username", "sunil");
editor.apply();
String user = prefs.getString("username", "");

Internal Storage

FileOutputStream fos = openFileOutput("data.txt", Context.MODE_PRIVATE);
fos.write(data.getBytes());
fos.close();

External Storage

File file = new File(getExternalFilesDir(null), "data.txt");
FileOutputStream fos = new FileOutputStream(file);
fos.write(data.getBytes());
fos.close();

SQLite Database

SQLiteDatabase db = openOrCreateDatabase("mydb", MODE_PRIVATE, null);
db.execSQL("CREATE TABLE IF NOT EXISTS users(id INTEGER, name TEXT)");
db.execSQL("INSERT INTO users VALUES(1, 'Sunil')");

Room Database (Recommended)

@Entity
public class User {
@PrimaryKey public int id;
public String name;
}
@Dao
public interface UserDao {
@Insert void insert(User user);
@Query("SELECT * FROM user") List<User> getAll();
}
@Database(entities = {User.class}, version = 1)
public abstract class AppDatabase extends RoomDatabase {
public abstract UserDao userDao();
}
