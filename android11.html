Chapter 11: Material Design 3

Material Design 3 (Material You)

Material Design 3 introduces a more expressive and personalized design system with dynamic color, improved accessibility, and modern components.

Material Design 3 Principles

  • Personalization: Dynamic color based on user's wallpaper
  • Expressiveness: More rounded corners, refined typography, and expressive motion
  • Accessibility: Better contrast ratios and touch targets
  • Consistency: Unified design language across platforms

// Check if dynamic color is available
public class ThemeUtils {
public static boolean isDynamicColorAvailable(Context context) {
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.S) {
return true; // Android 12+
}
return false;
}

public static void applyDynamicColors(Activity activity) {
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.S) {
DynamicColors.applyToActivityIfAvailable(activity);
}
}
}

// Apply dynamic colors in your activity
@Override
protected void onCreate(Bundle savedInstanceState) {
super.onCreate(savedInstanceState);
ThemeUtils.applyDynamicColors(this);
setContentView(R.layout.activity_main);
}

Migration from Material Design 2

Material Design 3 is backward compatible, but you should gradually migrate your existing components to take advantage of the new features.

  • Update your theme to inherit from Material3 themes
  • Replace old Material Components with new ones
  • Update color schemes to use the new color system
  • Implement dynamic color where appropriate
  • Test your app in both light and dark themes
  • Verify accessibility compliance
